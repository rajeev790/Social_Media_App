# Social Media App

## Overview

The Social Media App is a platform where users can create profiles, share posts, like and comment on others' posts, follow other users, and receive notifications. The app features user authentication, post tagging, and multimedia uploads. Built with Django for the backend and React.js for the frontend, this application aims to provide a robust social media experience.

## Features

- **User Authentication:** Register, log in, and manage user profiles.
- **Post Management:** Create, like, comment on, and tag posts.
- **Follow System:** Follow and unfollow other users.
- **Notifications:** Receive notifications for post interactions and follow requests.
- **Explore:** Discover trending content and users.
- **Media Uploads:** Upload and share images and videos.

## Technologies Used

- **Backend:** Django, Django Rest Framework
- **Frontend:** React.js, CSS
- **Database:** PostgreSQL
- **Storage:** AWS S3 (for media files)
- **Containerization:** Docker
- **Deployment:** Docker Compose
